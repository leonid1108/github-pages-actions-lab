Автоматический деплой сайта через GitHub Actions

Это тестовая статическая страница для лабораторной работы. Она публикуется на GitHub Pages автоматически после push в ветку main.

Цель: освоить настройку CI/CD для статического сайта.

Инструменты: GitHub, GitHub Actions, GitHub Pages, peaceiris/actions-gh-pages.

Работа выполнена в рамках изучения автоматического деплоя статического сайта.
